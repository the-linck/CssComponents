Alert

The Alert Component provides a way to show important static messages to the user.

Notice: If there must be elements for interaction, like buttons, then it must be used an Alert Dialog role instead.

This is a standart alert, dude.
Example: HTML inline code <div role="alert" title="A simple alert">  This is an alert, dude. </div>

Specific Types

We also provide four specialized types of alerts for using in specific cases in you application.
They are all chosen from the data-css-type attribute values:

  • info
    Just some information that may be useful to the user, like a hint.
  • success
    Some operation finished with no problem.
  • warning
    There mith be a problem
  • error
    Something gone wrong
Just to let you know.
It's all right.
But that may be dangerous.
OH MY! WHAT happened here?
Example: HTML inline code <div data-css-type="info" role="alert" title="A simple alert">  Just to let you know. </div> <div data-css-type="success" role="alert" title="A simple alert">  It's all right. </div> <div data-css-type="warning" role="alert" title="A simple alert">  But that may be dangerous. </div> <div data-css-type="error" role="alert" title="A simple alert">  OH MY! WHAT happened here? </div>
