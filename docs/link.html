Link

Links are used everywhere on the web, 'couse we couldn't leave a freature like this out of our library.
To style links, both for the native a element and for custom elements acting as links, you must declare the element as a link, with data-css-role="link" or role="link" (non native) attribute.

I'm a native link.  
I'm a custom link.
Example: HTML inline code <a data-css-role="link" href="#plain">  I'm a native link. </a> &emsp; <div role="link">  I'm a custom link. </div>

There's it, this is the simplest component of the library.
In case you're wondering why there's an &emsp; entity on the code, the reason is quite simple: to separate both links, because as they are inline content, by default they'd be too close to each other.

States

As seasoned developers would expect, there' 4 link states supported:

  • normal (default)
  • visited
  • selected (hover / focus / current)
  • active

You can force one of them by the data-css-state attribute:

Normal   Visited   Selected   Active
Example: HTML inline code <a data-css-state="normal" data-css-role="link" href="#">  Normal </a> &emsp; <a data-css-state="visited" data-css-role="link" href="#">  Visited </a> &emsp; <a data-css-state="selected" data-css-role="link" href="#">  Selected </a> &emsp; <a data-css-state="active" data-css-role="link" href="#">  Active </a>

Notice: It's also possible to force the selected state with the aria-current="true" attribute, but it's the only link state supported by ARIA up to now.

Accent Colors

As any other component, link can be used with your chosen accent color too:

No forced state   Normal   Visited   Selected   Active
Example: HTML inline code <a data-css-accent data-css-role="link" href="#">  No forced state </a> &emsp; <a data-css-accent data-css-state="normal" data-css-role="link" href="#">  Normal </a> &emsp; <a data-css-accent data-css-state="visited" data-css-role="link" href="#">  Visited </a> &emsp; <a data-css-accent data-css-state="selected" data-css-role="link" href="#">  Selected </a> &emsp; <a data-css-accent data-css-state="active" data-css-role="link" href="#">  Active </a>
